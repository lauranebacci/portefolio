Bootstrap est un framework développé par l'équipe du réseau social Twitter. Proposé en open source (sous licence MIT), ce framework utilisant les langages HTML, CSS et JavaScript fournit aux développeurs des outils pour créer un site facilement. Ce framework est pensé pour développer des sites avec un design responsive, qui s'adapte à tout type d'écran, et en priorité pour les smartphones. Il fournit des outils avec des styles déjà en place pour des typographies, des boutons, des interfaces de navigation et bien d'autres encore. On appelle ce type de framework un "Front-End Framework".

Le framework Bootstrap connait de nombreuses évolutions. En 2012, Bootstrap est classé comme le projet le plus populaire sur GitHub. Plus tard, sa deuxième version est sortie . Cette dernière comprend ce que l'on appelle la conception de sites Web adaptatifs, ce qui permet aux sites et applications Web utilisant Bootstrap de s'adapter dynamiquement au format des supports depuis lesquels ils sont accédés (ordinateur, tablette et smartphone). De plus, la version comporte des éléments de personnalisation supplémentaires pour rendre ces projets plus agréables à visualiser et travailler. Enfin, Bootstrap comprend une série de feuilles de styles, appelées LESS, qui sont englobées dans une même feuille de style : bootstrap.less. Ainsi, les développeurs débutants ou experts peuvent sélectionner les composants dont ils ont besoin en modifiant leur feuille de style principale. Ils apparaissent dans la version 4 du langage SASS, et permettent la manipulation de variables, de fonctions, d'opérateurs et de sélecteurs (un peu comme JavaScript ou PHP dans ce domaine).
